# 🚗 Java All-in-One OOP Concepts Program

## 📌 Overview

This project demonstrates **core Java programming concepts** in a single structured program.
It is designed for **students, beginners, and interview preparation**, covering almost all important Object-Oriented Programming (OOP) principles.

---

## 🎯 Concepts Covered

* ✅ Classes & Objects
* ✅ Encapsulation (private variables + getters)
* ✅ Inheritance
* ✅ Abstraction (abstract class)
* ✅ Polymorphism (method overriding & dynamic binding)
* ✅ Interface implementation
* ✅ Exception Handling (try-catch)
* ✅ Collections Framework (ArrayList)
* ✅ User Input (Scanner)

---

## 🏗️ Project Structure

* `Vehicle` → Abstract base class
* `Car` & `Bike` → Derived classes
* `Service` → Interface
* `MainProgram` → Driver class

---

## ⚙️ How It Works

1. Vehicles are created using inheritance.
2. Different vehicle types override methods like `mileage()` and `start()`.
3. Polymorphism is demonstrated using a common reference (`Vehicle`).
4. Interface (`Service`) adds additional behavior.
5. ArrayList stores multiple vehicle objects.
6. Exception handling manages invalid user input.

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/java-oop-concepts.git
```

2. Compile the program:

```
javac MainProgram.java
```

3. Run:

```
java MainProgram
```
